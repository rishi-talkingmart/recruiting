# workflow-frontend
Frontend components for the workflow tooling including UWES

# Setup-intstructions
- Clone the repo
- Setup the environment variables as given below
- Install Yarn if not already installed
- After that run the folowing commands to run the code
```shell
yarn
yarn run serve
```
